# 빨간줄 발생 1
## 에러 내용
```
Type annotations can only be used in TypeScript files
```

## 해결 방법
```
vscode -> setting -> [javascript > Validation: Enable] 해당 설정 해제

* 이 방법 말고도 "Flow Language Support" 해당 extention 설치해서 해결 가능. 아래 스택오버플로우 참고 바람.
```
